The Stable Seas Maritime Security Index is an interactive report that provides a holistic analysis of maritime security issues in sub-Saharan Africa.

The interactive report format is only available on modern desktop browsers such as Chrome, Safari and Firefox. It is not currently supported on mobile platforms or on Internet Explorer.

For the full experience, please visit www.stableseas.orgs on a desktop browser. (If you are on a desktop browser please maximize your screen and click the link.) An executive summary of our team's findings can be found below.

Stable Seas is a collaborative project created by One Earth Future programs Oceans Beyond Piracy, Secure Fisheries and OEF Research.
